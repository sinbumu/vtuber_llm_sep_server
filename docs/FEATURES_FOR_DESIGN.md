# FEATURES_FOR_DESIGN.md — 디자이너 전달용 기능 카탈로그

## TL;DR (10줄)
1) 이 프로젝트는 **음성 기반 AI 동반자** 경험을 제공합니다.  
2) LLM/음성인식/음성합성/Live2D가 하나의 파이프라인으로 연결됩니다.  
3) **웹 UI**와 **데스크톱 클라이언트** 2가지 모드를 지원합니다.  
4) 데스크톱 클라이언트는 **투명 배경/항상 위 펫 모드**를 지원합니다.  
5) **실시간 음성 대화**와 **대화 중 끊기(인터럽트)** 기능이 핵심입니다.  
6) Live2D 표정/감정 매핑으로 **캐릭터 연출**이 가능합니다.  
7) **카메라/스크린 인식** 같은 시각 입력이 가능합니다.  
8) 다양한 LLM/ASR/TTS 엔진을 **설정 파일로 교체**할 수 있습니다.  
9) 오프라인 구성이 가능하지만, 일부 TTS는 클라우드가 필요합니다.  
10) Live2D **샘플 모델은 별도 라이선스**라 상용 제품은 교체 권장입니다.

---

## 1) 핵심(대화)

### 음성 기반 대화(실시간)
- 사용자 가치: 손을 쓰지 않고 자연스럽게 대화 가능
- UI 컨트롤: 마이크 시작/중지 버튼, 입력 레벨 표시, 대화 시작 버튼
- 권한: 마이크
- 적용: **MVP**

### 대화 중 끊기(인터럽트)
- 사용자 가치: 사용자가 말하면 AI가 바로 멈춤
- UI 컨트롤: “자동 끊기” 토글, 민감도 슬라이더
- 권한: 마이크
- 적용: **MVP**

### 대화 로그 지속 저장
- 사용자 가치: 이전 대화 이어하기
- UI 컨트롤: 대화 기록 목록, 검색, 삭제
- 권한: 로컬 저장소
- 적용: **MVP**

### AI 선제 발화(능동 발화)
- 사용자 가치: AI가 먼저 말 걸기
- UI 컨트롤: 토글, 빈도 슬라이더, 방해 금지 시간대
- 권한: 없음
- 적용: **심화**

---

## 2) 음성(ASR/TTS)

### 음성 인식(ASR)
- 사용자 가치: 말하면 텍스트로 인식
- UI 컨트롤: 언어 선택, 민감도, 입력 장치 선택
- 권한: 마이크
- 적용: **MVP**

### 음성 합성(TTS)
- 사용자 가치: AI가 자연스럽게 말함
- UI 컨트롤: 음성 엔진 선택, 보이스 선택, 속도/피치
- 권한: 네트워크(클라우드 TTS 사용 시)
- 적용: **MVP**

### 음성 번역(대화 언어/음성 분리)
- 사용자 가치: 텍스트는 A언어, 음성은 B언어 가능
- UI 컨트롤: 번역 토글, 원본/목표 언어 선택
- 권한: 네트워크(번역 서버 사용 시)
- 적용: **심화**

---

## 3) 캐릭터/연출(라이브2D)

### Live2D 캐릭터 표시
- 사용자 가치: 캐릭터가 살아있는 느낌 제공
- UI 컨트롤: 모델 선택, 스케일/위치/투명도
- 권한: 파일 접근(모델 로드)
- 적용: **MVP**

### 감정/표정 매핑
- 사용자 가치: 말투/감정에 맞는 표정 연출
- UI 컨트롤: 감정 맵 편집, 표현 강도 슬라이더
- 권한: 없음
- 적용: **MVP**

### 내면 생각 표시(무음 텍스트)
- 사용자 가치: 말하지 않는 생각을 텍스트로 보여줌
- UI 컨트롤: “생각 표시” 토글, 표시 위치/색상
- 권한: 없음
- 적용: **Nice-to-have**

### 터치/드래그 반응
- 사용자 가치: 캐릭터와 직접 상호작용
- UI 컨트롤: 터치 반응 토글, 감도 슬라이더
- 권한: 없음
- 적용: **Nice-to-have**

---

## 4) 운영/설정/확장

### 엔진 교체(LLM/ASR/TTS)
- 사용자 가치: 성능/비용/오프라인 요구에 맞춤
- UI 컨트롤: 엔진 드롭다운, API Key 입력, 테스트 버튼
- 권한: 네트워크(클라우드 사용 시)
- 적용: **MVP**

### 프리셋(캐릭터/설정 저장)
- 사용자 가치: 캐릭터/설정을 손쉽게 전환
- UI 컨트롤: 프리셋 목록, 저장/불러오기
- 권한: 로컬 저장소
- 적용: **MVP**

### MCP 도구/확장(외부 도구 연결)
- 사용자 가치: 웹검색/도구 사용 등 기능 확장
- UI 컨트롤: 도구 리스트, 권한 승인, 활성/비활성
- 권한: 네트워크/파일 등 도구별 상이
- 적용: **심화**

## 주의(라이선스/리스크)
- Live2D **샘플 모델은 별도 라이선스** → 상용 제품은 교체 권장
- 클라우드 TTS/ASR은 네트워크 차단 이슈 가능
